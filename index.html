Part of the vacation portion of the app involves some determining with numbers, specifically the age.
I made it so if you type in the word (like one, two, three, etc.), it will actually take in those words as a number, because the program converts those specific words into numbers using 'if' and 'else if' statements.

       if (age == "one" || age ==                "one".toUpperCase()) {
                  age = 1;
       }

This means if you type "one" or "ONE" into the age box, it will set the age to BE the number 1, and the program will decide which is the best location suited for the user.

 == MATH GAME APP PORTION ==
A lot goes into this portion of the app. I basically had to make a functioning calculator to randomly generate two numbers, and then take those numbers and the operator of the equation, and make the program SOLVE the equation AND get it to take the result/answer and round it to the nearest tenth, because sometimes when you calculate stuff with a lot of decimals, it might give a number like 2.613133341, so I made the program (using Math.round) round the answer, to (using the example number that I just used) 2.6.

Based on the difficulty that the user chooses for the game, the program will generate two random numbers and throw them into the equation, and the user has to try and solve the equation.


(the actual code is much different than this)

var firstNumber = randomNumber(1,7);
var secondNumber = randomNumber (1, 7);
var operator = randomNumber(1, 4);
var answer;

if (operator == 1) {
operator = "+";
} else if (operator == 2) {
operator = "-";
} else if (operator == 3) {
operator = "*";
} else if (operator == 4) {
operator = "/";
}

answer = firstNumber (+, -, * or /) secondNumber

We use 'if' statements to determine which operator the program should use in order to solve the equation, for example of the operator is a plus sign, then it will read what is in that if statement and add the two numbers together.

There's a lot more to this side of the app, including scoring, the points system, questions answered and maximum amount of questions. Happy mathing :).
